# Pointless Cheat Sheet — Sporting "Multiple Winners"

Obscure-but-safe answers for sports categories built around winning something
*more than once*. Each list favours **correct + low-scoring** picks (good on the
buzzer) over the obvious household names.

---

## 🎱 Snooker — Multiple World Championship winners *(since 1950)*

| Answer | Wins | Note |
|---|---|---|
| **John Pulman** | multiple | Challenge-match era (1960s) — the gold-standard "who?" answer |
| **John Spencer** | 3 | 1969, 1970, 1977 |
| **Ray Reardon** | 6 | 1970, 1973–76, 1978 |
| **Alex Higgins** | 2 | 1972, 1982 |
| **Fred Davis** | multiple | Titles across the 1950s |

*Crucible-era only (since 1977)?* Drop to **John Higgins (4)**, **Mark Williams
(3)**, **Mark Selby (4)** — that's the whole obscure pool.
